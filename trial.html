APB Analysis

The Annular Pressure Buildup analysis module evaluates APB due to temperature changes during circulation, production, and worst case discharge. The module automatically generates the multi-annular interactive APB which often drives the critical burst and collapse loads.

Additionally, the APB module provides the average pressure increase per unit temperature change (psi/°F), an intuitive measure of the thermo-mechanical stiffness of the annulus for a given wellbore configuration.

×

Allowable APB

Allowable Annular Pressure Buildup is the maximum additional pressure that can be tolerated by an annulus before structural compromise of the tubulars.

The allowable APB for a string is the minimum of the allowable APB for Inner String Collapse & allowable APB for Outer String Burst.

The Allowable APB module can be used in conjunction with its APB analysis counterpart to model multi-annular interaction.

×

Wellhead Movement

The wellhead movement module follows each stage of well construction and production via an intuitive interface, and presents wellhead forces and displacements, during all stages.

It provides lockdown ring ratings, an important feature in subsea wellheads with stiff intermediate casings.

The strength of the structural pile is determined from the undrained soil shear strength data and the API RP 2GEO equations.
