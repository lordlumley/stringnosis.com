WHY STRINGNOSIS®

It reflects the logic and information flow of actual well design. Analytical results provides complete knowledge of each casing, liner or tubing string.

STRINGNOSIS® FEATURES

WHICH VERSION IS FOR YOU?

STRINGNOSIS® is for engineers to design, analyse and optimize any type of well. The user interface and functionalities are intuitive, with customizable standard load packages, so that even standard well designs are easy and quick.

The software is available in two versions, core and advanced. The software reflects the type of wells you work with.

Working Stress Design methodology is features in both core and advanced versions. The advanced version uniquely includes two additional methodologies, Limit State Design & Reliability Based Design.
